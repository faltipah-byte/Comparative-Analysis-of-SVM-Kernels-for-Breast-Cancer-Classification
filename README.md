# Comparative-Analysis-of-SVM-Kernels-for-Breast-Cancer-Classification
Proyek ini bertujuan untuk membandingkan berbagai jenis kernel pada algoritma Support Vector Machine (SVM) guna menentukan kernel terbaik dalam melakukan klasifikasi kanker payudara. Dataset yang digunakan berasal dari dataset bawaan Python.

Berdasarkan hasil penelitian, diperoleh bahwa kernel linear menghasilkan akurasi tertinggi sebesar 90% dibandingkan kernel lainnya. Namun, hasil ini masih memiliki keterbatasan karena belum dilakukan optimasi hyperparameter secara menyeluruh, sehingga belum dapat dipastikan bahwa kernel linear merupakan model terbaik secara optimal.
